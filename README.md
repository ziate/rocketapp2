# نظام إدارة توصيل الطلبات

مشروع لوحة تحكم متكاملة لإدارة الطلبات والعملاء والمناديب والمحافظات والمناطق وأنواع الطلبات.

## المميزات
- لوحة تحكم عربية بالكامل مع قائمة جانبية واضحة.
- إدارة المحافظات والمناطق مع أسعار التوصيل الافتراضية.
- إدارة العملاء مع إمكانية تخصيص أسعار التوصيل حسب المنطقة لكل عميل.
- إدارة مناديب التوصيل (إضافة/تعديل/حذف).
- إدارة أنواع الطلبات.
- إنشاء طلبات مع بيانات المستلم وخيارات التحصيل.
