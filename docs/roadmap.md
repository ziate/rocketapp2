# اقتراحات لتطوير المشروع قبل الإطلاق

## 1) تحسينات أساسية للتشغيل
- نظام صلاحيات (أدوار وصلاحيات) للمستخدمين.
- إدارة حالات الطلب بدقة (تاريخ التحديث، من قام بالتغيير).
- تسجيل نشاطات المستخدمين (Audit Log).

## 2) تجربة المستخدم
- فلاتر بحث متقدمة للطلبات والعملاء والمناديب.
- دعم استيراد/تصدير CSV.
- لوحات تقارير تفصيلية حسب المحافظة والمنطقة.

## 3) عمليات التوصيل
- آلية إسناد الطلبات للمناديب آليًا أو يدويًا.
- إشعارات SMS/Email للعميل والمندوب.
- تتبع حالة الطلب في الزمن الحقيقي (إن توفر مزود).

## 4) إعدادات التسعير
- أسعار خاصة حسب نوع الطلب.
- حد أدنى/أقصى لقيمة التحصيل.
- قواعد خصومات للعملاء المميزين.

## 5) الأداء والبنية
- إضافة طبقة Services/Repositories لكل منطق عمل مركزي.
- Cache للاستعلامات المتكررة (المناطق، المحافظات).
- اختبارات وحدات واختبارات تكامل للطلبات.
