# قائمة تحقق للاستضافة المشتركة

## قبل الرفع
- إنشاء مشروع Laravel جديد محليًا عبر Composer.
- نسخ ملفات هذا المستودع إلى مشروع Laravel.
- تنفيذ `composer install` محليًا لتجهيز مجلد `vendor`.
- تجهيز قاعدة البيانات وإضافة بياناتها في `.env`.

## بعد الرفع
- التأكد من إعداد Document Root على مجلد `public`.
- ضبط صلاحيات `storage` و `bootstrap/cache` إلى 775 أو 755 حسب الاستضافة.
- التأكد من رفع ملف `.env` وعدم تعريضه للعرض.
- تشغيل `php artisan migrate --force` عبر SSH إن كان متاحًا.
- اختبار لوحة التحكم عبر `/`.

## إعدادات مقترحة للأمان
- تفعيل HTTPS.
- تقييد الوصول إلى `/storage` إن لزم.
- تعطيل عرض الأخطاء في الإنتاج عبر `APP_DEBUG=false`.
